## 语音合成并播放（流式模式）
本示例展示了如何合成指定文本的语音，获取流式返回音频播放。本示例同时展示了如何在流式回调中保存音频到文件。

| **数字人场景** | 新闻播报 | *新闻等场景，通过语音合成进行文本信息的播报* |
