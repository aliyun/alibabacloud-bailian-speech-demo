## 批量音视频文件语音识别（批量模式）
本示例展示了如何大批量的提交存储于OSS等云存储中的音视频文件链接，并调用文件转写服务实现批量语音识别的过程。

相对于实时语音识别，录音文件识别提供了更多语音格式、以及信息更丰富的结果供用户选择。同时批量音视频文件语音识别，相对于实时语音识别更适合时效性要求不高批量的任务处理。
